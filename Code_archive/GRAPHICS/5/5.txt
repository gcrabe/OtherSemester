Задание 5. Построение образа абстрактного изображения
Создайте проект, в котором изобразите график вещественнозначной функции в прямоугольной области в окне:
Опишите вещественнозначную функцию f(x)f(x), для которой вы изображаете график, как функцию в отдельном файле function.cppfunction.cpp. В этом же файле опишите функцию fexists(x)fexists(x), возвращающую true, когда функция ff может быть вычислена для аргумента xx и false - в противном случае. Подключите описанные функции к вашему проекту.
При инициализации формы задайте значения VxVx, VyVy, VcxVcx, VcyVcy для первоначального кадра с изображением функции в картинной плоскости, совмещаемого с прямоугольником на экране.
Организуйте отрисовку графика функции ff в прямоугольной области видимости на форме приложения. График необходимо отрисовывать как набор из WxWx последовательных отрезков, где WxWx - ширина прямоугольной области для вывода графика. Каждый очередной отрезок должен соединять точки (x?,y?)(x?,y?) и (x?+1,y??)(x?+1,y?), где y?y? и y??y? вычисляются в соответствии с заданной функцией (см. методические рекомендации). 
С помощью алгоритма удаления невидимых линий, реализованном при выполнении предыдущего задания, обеспечьте удаление частей отрезков, составляющих график, не попавших в область видимости. 
Создайте обрамление области видимости. Добавьте к области видимости координатную сетку. Сетка должна состоять из LL вертикальных и KK горизонтальных линий (LL и KK - параметры формы с некоторыми начальными значениями). Вертикальные линии должны делить всю область видимости на L+1L+1 равных частей, горизонтальные линии - на K+1K+1 равных частей. Рядом с каждой вертикальной линией сетки у верхнего края области видимости нужно вывести значение соответствующей координаты xx картинной плоскости. Рядом с каждой горизонтальной линией сетки у левого края области видимости нужно вывести значение соответствующей координаты yy картинной плоскости.
Добавьте реакции на нажатие клавиш I, O, K, L для масштабирования графика отдельно по каждой из осей относительно центра области видимости и A, D, W, S для смещения графика в области видимости. При выполнении этих операций координатная сетка должна оставаться на месте, но должны пересчитываться значения, выводимые рядом с координатной сеткой. Шаг сдвига при смещении графика должен равняться 1 пикселу и не должен зависеть от масштаба вывода графика.
Добавьте реакции на нажатие клавиш T, G, F, H для быстрого смещения графика в области видимости. Шаг сдвига при смещении графика не должен зависеть от масштаба вывода графика.
Добавьте реакции на нажатие клавиш Q, E, Z, C для увеличения/уменьшения количества вертикальных/горизонтальных линий координатной сетки.
Добавьте реакцию на нажатие клавиши для восстановления первоначального изображения графика.
Область видимости вместе с изображением в ней должны пропорционально масштабироваться при увеличении/уменьшении размеров окна.
