Разработка SQL кода БД
Создать схему базы данных  из задания 1 в виде набора DDL-инструкций и реализовать ее в базу данных средствами СУБД Microsoft SQL Server. Написать код заполнения данными и очистки таблиц базы данных. 

Рекомендация: при выполнении задания 2 предпочтительно использовать язык SQL  для создания схемы базы, а не средства автоматической генерации таблиц и связей между ними. Это связано с тем, что настройки СУБД, выставленные по умолчанию для автоматизированных средств проектирования создают в дальнейшем проблемы при модификации схемы.

Строго запрещается использование в названиях отношений, атрибутов, и любых других объектов в базу данных кириллицы, только латинский алфавит. 

Заполняется база данных информацией преимущественно на русском языке.

 Привести примеры использования запросов Insert и Delete (удаление всех записей из таблицы, удаление отдельных записей из таблицы)
