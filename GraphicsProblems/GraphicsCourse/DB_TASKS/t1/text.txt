Построение реляционной модели данных
Разработать проект базы данных (на основе предложенных примеров), содержащей отношения с заданными полями и ограничениями на них


Перечень предметных областей для лабораторных занятий

1. Гостиница

Справочники: Номера, Услуги, Клиенты

Функции: Ведение справочников, поселение и выселение клиентов, бронирование мест, учёт оказанных услуг

Выходные документы: Счёт за проживание и услуги, Список проживавших на момент времени, Список номеров, Прейскурант услуг

2. Успеваемость студентов

Справочники: факультеты, кафедры, студенты, предметы

Функции: ведение справочников, учёт успеваемости студентов

Выходные документы: Ведомость успеваемости по группе студентов, Переводной приказ, Аналитические отчёты об успеваемости

3. Больница

Справочники: Пациенты, Болезни, Палаты, Врачи, История болезни

Функции: Ведение справочников, приём пациента, ведение истории болезни, выписка.

Выходные документы: Список пациентов, Карточка больного, Аналитические отчеты по сезонным подъемам заболеваемости, по возрастным группам.

4. Склад

Справочники: Группы товаров, Товары, Производители, Поставщики

Функции: ведение справочников, учёт прихода и продаж товаров

Выходные документы: Отчёт по наличию товаров на складе по группам, Отчёт по продажам по группам, Счёт-фактура

5. Ресторан

Справочники: Продукты, Блюда, Заказы, Дисконт

Функции: Ведение справочников, хранение рецептов, расчёт себестоимости блюда, приём заказов

Выходные документы: Меню, Счёт заказа, Отчёт по продуктам на складе, Заказы за период.

6. Транспортные перевозки

Справочники: Машины, Водители, Клиенты

Функции: Приём заказов на грузоперевозки

Выходные документы: Счёт, Отчёт по заказам за период, Отчёт по имеющимся в наличии свободным машинам

8. Отдел кадров

Справочники: Оргструктура предприятия, Должности, Сотрудники.

Функции: Приём, перевод, увольнение сотрудников

Выходные документы: Отчёт по штатному составу (по подразделению), бланки приказов на приём, перевод, увольнение

9. Зарплата

Справочники: Сотрудники, отделы, должности

Функции: расчёт зарплаты – почасовая оплата, начисления, удержания, налоги.

Выходные документы: Платёжная ведомость, расчётный листок

10. Ателье

Справочники: Клиенты, Материалы, Поставщики, Модели одежды

Функции: Оформление заказов на материалы от поставщиков, заказов на работы от клиентов

Выходные документы: Счёт-фактура, счёт клиенту, отчёт по выполненным заказам за период

11. Библиотека

Справочники: Авторы, Книги, Жанры, Издательства, Читатели

Функции: Учёт поступлений книг, Поиск книг по критериям, Учет выданных книг

Выходные документы: Карточка книги, Новинки, Поиск книг, Формуляр читателя.

12. Цветочный магазин

Справочники: Цветы, Букеты, Клиенты.

Функции: Учет продаж отдельных цветов и букетов, расчёт прогрессивных скидок постоянным клиентам

Выходные документы: Счёт, Отчёт по продажам за период, Отчёт по продажам по видам цветов.

13. Кадровое агенство

Справочники: Работодатели, вакансии, соскатели

Функции: Учёт вакансий от работодателей, формализованных резюме от соискателей, подыскание подходящей вакансии для соискателя

Выходные документы: Список текущих вакансий по критериям (зп, опыт, должность). Резюме соискателя

14. Оплата в ТСЖ 

Справочники: Дом, Жильцы, Услуги, Счетчики.

Функции: Ежемесячный расчёт стоимости услуг на основании данных счётчика и числа проживающих

Выходные документы: Счёт для каждого жильца. Информация о задолжниках.

15. Агентство недвижимости

Справочники:  Улицы, Дома, Типы домов, Типы квартир и т.п.

Функции: Хранение подробной информации о сдающемся и продающемся жилье – дома, квартиры (площадь, условия), поиск квартиры по критериям

Выходные документы: Отчёт «найденные квартиры по критериям», Карточка квартиры.

16. Ремонт машин

Справочники:  Клиенты, Детали, Услуги, Машины, Работники

Функции: Приём заказов, Учёт производственных операций, Формирование счёта за услуги, Назначение исполнителей работ

Выходные документы: Счёт, Отчёт по работам за период, Прайс-листы на услуги и детали

17. Рецепты

Справочники: Продукты, Блюда, Страны, Группы блюд

Функции: Составление рецептов, расчёт калорийности и себестоимости блюда

Выходные документы: Рецепт, Количество рецептов по группам, Калорийность продуктов, Подбор меню по географическому принципу

18. Авиакассы

Справочники: Рейсы, Самолёты, Аэропорты, Компании

Функции: Ведение справочников, ведение расписания, подбор рейса, выдача билета

Выходные документы: Расписание, Билет, Отчёт по проданным билетам за период, Отчёт по рейсу

19. Туристическая фирма

Справочники: Страны, Маршруты, Достопримечательности, Компании

Функции: Хранение информации о маршрутах, подбор маршрута для клиента, приём заказа

Выходные документы: Отчёт – подходящие маршруты по критериям (цена, страна, длительность,...), бланк заявки, отчёт по заказам. Аналитический отчет о популярности маршрутов, достопримечательностей и т.д.

20. Тестирование

Справочники: Дисциплины, вопросы, ответы, студенты

Функции: хранение тестов с вариантами ответов по разным дисциплинам, регистрация студентов, хранение истории тестирования

Выходные документы: Распечатка теста для бумажного тестирования, Справка о результате тестирования, Отчёт о проведённых тестированиях за период

21. Расписание маршрутов движения транспорта

Справочники: Остановки, Автобусы, Компании, Маршруты.

Функции: Составление расписания городских автобусов

Выходные документы: Расписание автобуса номер N, Маршрутный лист водителя, Заказ на ремонт.

22. Блог

Справочники: Авторы, Темы, Теги для поиска информации, авторы комментариев

Функции: Поиск авторов, Поиск тем, Добавление тегов, Добавление новых тем и комментариев

Выходные документы: Аналитические отчеты по популярности тем, авторов, активных комментаторов.

23. Журнал учета заявок на обслуживание техники

Справочники: Клиенты, Задачи, Подразделения

Функции: Приём заказов, отслеживание их выполнения

Выходные документы: Форма заявки, Отчёт по выполенным/невыполненным заявкам за период, Заявки по клиентам

24. Тренажёрный зал

Справочники: Залы, Виды спорта, Тренера, Клиенты

Функции: Ведение справочников, составление расписания

Выходные документы: Расписание по видам спорта, график загрузки тренера, график загрузки зала
