Задание 7. Отсечение многоугольников

Создайте копию проекта, реализованного при выполнении задания 4. Внесите в проект изменения, описанные в методических рекомендациях:
  1. Опишите структуру данных для хранения списка многоугольников.
  2. Реализуйте процесс отрисовки многоугольников из описанной структуры данных.
  3. Организуйте отрисовку изображения, состоящего из набора многоугольников, вершины которых заданы в правой системе координат. Для этого организуйте чтение файла в новом формате. Входной файл может содержать пустые строки, строки комментариев (строки начинающиеся с символа ##), а также строки с командами:
    1. frame Vcx Vcy Vx Vyframe Vcx Vcy Vx Vy - команда выделения кадра. Если команда встречается в файле более одного раза, то каждый последующий экземпляр команды отменяет действие предыдущей;
    2. polygon n x1 y1 x2 y2 ... xn ynpolygon n x1 y1 x2 y2 ... xn yn - команда, задающая многоугольник с nn вершинами, в котором числа xixi или yiyi - координаты ii-ой вершины многоугольника в порядке обхода вершин по часовой (или против часовой) стрелки.
  4. Реализуйте алгоритм отсечения многоугольников, соответствующий вашему варианту. Основная функция, реализующая алгоритм должна возвращать многоугольник - результат отсечения. Сигнатура функции отсечения приведена в разделе 2 методических рекомендаций.

Методические рекомендации.

Вариант 1: алгоритм Лианга-Барски (читать алгоритм);
Вариант 2: алгоритм Кируса-Бека (читать алгоритм);
Вариант 3: алгоритм Мэл'от (читать алгоритм);
Вариант 4: алгоритм Сазерленда-Ходгмана (читать алгоритм).
